
# Weather App

This is the Weather App in Which you can See Detail Weather of your Location:                             
1. Temperature
2. Type Of weather(Clear Sky,Rainy,Cloudy etc....)
3. Min and Max Tempurature
4. Wind Speed
5. Humidity 
6. SunRise and SunSet Time
